# 레시피 소개 리액트 앱

## App 의 구조

```md #
    src/
     ├─── App
     ├─── components/ (UI요소를 리턴하는 컴포넌트파일)
     │				│
     │				├─── Home.js 메인컴포넌트
     │				├─── ListGroup.js 카테고리 컴포넌트
     │				├─── Recipe.js 리스팅컴포넌트
     │				├─── Title.js 제목컴포넌트
     │				└─── Detail.js 상세정보 컴포넌트
     │
     └─── routes /  (라우팅 설정 파일)
      				│
      				└─── Router.js 라우터와 링크설정
```
